matplotlib

matplotlib is a Python library to assist in displaying data easily and elegantly.

Pip install matplotlib and add 'import matplotlib' and 'import matplotlib.pyplot as plt' to views.py to get started.

Energy Consumption by Source

Data source: U.S. Energy Information Administration

  • Twelve API requests (one per energy source)
  • Energy consumption on the y-axis; Year on the x-axis
  • matplotlib automatically changes the colors for each data set

Weather Forecasts

Data source: Weather Underground

  • Temperature on y-axis; Hour on x-axis
  • Marker color is related to temperature
  • Marker size is related to humidity (larger = more humid)
